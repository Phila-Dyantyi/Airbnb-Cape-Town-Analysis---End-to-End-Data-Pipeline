# Airbnb Cape Town Analysis - End to End Data Pipeline
Analysing Airbnb listings in Cape Town and exploring correlations between listing ratings and weather conditions.

An end-to-end data engineering project analysing Airbnb listings in Cape Town alongside daily weather data to uncover correlations between weather conditions and listing ratings. The pipeline covers data acquisition via the Open-Meteo API, ETL processing in Python, and data transformation in SQL Server using staging tables, views, a dimensional data warehouse, and pipeline logging. Insights are delivered through a Power BI dashboard.

The analysis focuses on:

- Distribution and quality of Airbnb listings across Cape Town neighbourhoods
- Weather conditions in Cape Town during June and September 2025
- Potential correlations between weather patterns and listing ratings
- Additional insights surfaced through exploratory data analysis
